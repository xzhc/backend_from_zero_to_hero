## 内容：

### 1. 数据结构

变量在内存中占据一块空间，这块空间有一个**地址**（类似门牌号）。
指针就是用来**存储这个地址**的变量。

> 📖 **新手友好补充**
>
> 🔍 **生活类比：门牌号 vs 房间里的东西**
>
> 想象你住在一个大楼里：
> - **变量** = 房间里的东西（比如一个苹果）
> - **内存地址** = 房间的门牌号（比如 `101`）
> - **指针** = 一张写着门牌号的纸条
>
> 当你把"门牌号纸条"给别人，别人就能**找到那个房间**，拿到或修改里面的苹果——这就是指针的核心价值！

---

### 2. 指向所引用的变量的存储地址

指针变量存储的是另一个变量的**内存地址**，通过这个地址可以间接访问或修改原变量的值。

> 📖 **新手友好补充**
>
> 💡 **为什么需要指针？**
>
> Go 中函数传参默认是**值传递**（复制一份），这意味着函数内部修改参数不会影响外部的原变量。
>
> 有了指针，我们可以**传递地址**，让函数直接操作原来的变量，实现"函数内部修改，外部也生效"的效果。
>
> ```go
> // 没有指针：函数内修改，外部不变
> func addOne(n int) {
>     n = n + 1  // 只修改了副本，外部的 x 不变
> }
>
> // 有了指针：函数内修改，外部也变
> func addOnePtr(n *int) {
>     *n = *n + 1  // 通过地址修改原变量
> }
> ```

### 3. `&`、`*int`、`*ptr` 三个核心符号

| 符号 | 名称 | 作用 | 示例 |
|------|------|------|------|
| `&`  | 取地址符 | 获取变量的内存地址 | `&i` → 变量 `i` 的地址 |
| `*int` | 指针类型 | 声明"指向 int 的指针"类型 | `var p *int` |
| `*ptr` | 解引用 | 通过指针访问/修改其指向的值 | `*p = 2` |

> 📖 **新手友好补充**
>
> 📝 **最小可运行示例**
>
> ```go
> package practice
>
> import "fmt"
>
> func TestPointer(t *testing.T) {
>     i := 42
>
>     // & 取地址：获取变量 i 的内存地址
>     p := &i
>     fmt.Println("i 的地址:", p)   // 输出类似：0xc0000b4008
>     fmt.Println("i 的值:", *p)    // 解引用：输出 42
>
>     // *ptr 解引用赋值：通过指针修改原变量
>     *p = 100
>     fmt.Println("修改后 i =", i)  // 输出：100（i 被真实修改了！）
> }
> ```
>
> ⚠️ **新手常见误区**
>
> 1. **`*` 的含义随语境变化**：
>    - 在**类型声明**时：`var p *int` → `*int` 表示"指向 int 的指针类型"
>    - 在**表达式**中：`*p` → 表示"解引用，取出指针指向的值"
>    - 初学者很容易把这两种用法混淆，记住：**类型里是声明，表达式里是操作**。
>
> 2. **对 nil 指针解引用会 panic**：
>    ```go
>    var p *int  // p 是 nil，没有指向任何地址
>    fmt.Println(*p)  // ❌ 程序崩溃！runtime panic
>    ```
>    使用指针前，务必确认它不是 `nil`。
>
> 3. **`&` 和 `*` 互为逆操作**：
>    - `&i` 取得 i 的地址
>    - `*(&i)` 再解引用，得到的还是 i 的值
>    - 记忆口诀：`&` 是"取地址"，`*` 是"取值"
